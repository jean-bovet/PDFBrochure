<h2>PDFBrochure 0.5</h2>

<p>First public release. PDFBrochure turns any PDF into a saddle-stitch booklet ready to print double-sided and fold &mdash; two source pages laid out per sheet, in the right order.</p>

<ul>
    <li><b>Drop a PDF, get a brochure.</b> Drag-and-drop a PDF onto the window (or click to pick one). Pick a sheet size and page placement, see a live preview update as you change them, then click <b>Create Brochure</b> to write <code>&lt;input&gt;-brochure.pdf</code> next to the source. The app never overwrites an existing file &mdash; it auto-suffixes <code>-2</code>, <code>-3</code>, &hellip; on collision.</li>
    <li><b>Smart paper-size default.</b> A4 input becomes A4 landscape (two A5 pages per sheet); A5 input becomes A4 landscape at native size; other sizes default to <i>auto</i> (input width doubled).</li>
    <li><b>Imposition rules in plain English.</b> For an N-page document padded up to a multiple of 4, sheet <i>k</i> holds <code>(N&minus;2k+2 | 2k&minus;1)</code> on the outer side and <code>(2k | N&minus;2k+1)</code> on the inner side. Print double-sided (flip on short edge), nest the sheets, fold along the spine, staple. Pages beyond the original count are left blank.</li>
    <li><b>Headless CLI included.</b> A small <code>pdfbrochurecli</code> tool ships in the repo's build for scripting and tests.</li>
    <li><b>Auto-update support.</b> Future releases land via Sparkle's "Check for Updates&hellip;" menu item, with EdDSA-signed update artefacts.</li>
</ul>

<p><i>Apple Silicon only. Requires macOS 13 or later.</i></p>
