# PDFBrochure

A small SwiftUI macOS app that turns any PDF into a saddle-stitch booklet PDF
(two input pages laid out per sheet, in the order needed for double-sided
printing and folding).

The flow is two screens:

1. **Drop** — drag a PDF onto the window (or click to pick one).
2. **Layout + preview** — pick a sheet size and page placement; the imposed
   PDF re-renders in a live preview. One click writes
   `<input>-booklet.pdf` next to the source (auto-suffixed `-2`, `-3`, … if
   the name is taken — never overwrites an existing file).

A small headless `bookletcli` target is built from the same imposition code
for scripting and tests.

## Repo layout

```
.
├── project.yml                          # XcodeGen spec — 2 targets
├── README.md
└── Sources/
    ├── App/BookletApp.swift             # SwiftUI UI + view model
    ├── CLI/main.swift                   # bookletcli entry point
    └── Imposition/BookletMaker.swift    # shared imposition logic
```

`Booklet.xcodeproj/`, `Sources/Info.plist`, and `Sources/Booklet.entitlements`
are generated by XcodeGen from `project.yml` and are gitignored. Run
`xcodegen generate` after cloning.

## Build & run

```sh
brew install xcodegen          # one-time
xcodegen generate              # creates Booklet.xcodeproj from project.yml
open Booklet.xcodeproj         # …and hit Run, or:

xcodebuild -scheme Booklet    -configuration Debug   build
xcodebuild -scheme bookletcli -configuration Release build
```

The built app lives under `~/Library/Developer/Xcode/DerivedData/Booklet-*/Build/Products/`.

> Don't pass `-derivedDataPath` to point at a folder under iCloud-managed
> directories like `~/Desktop` or `~/Documents`: the iCloud file-provider
> attaches xattrs that make codesign refuse to sign the resulting `.app`.

### Headless usage

```
bookletcli <input.pdf> <output.pdf> [auto|a4|a3|letter|tabloid] [fit|fill|original]
```

## How the imposition works

For an N-page document padded up to a multiple of 4, sheet *k* (1-indexed)
holds:

| side  | left half      | right half     |
|-------|----------------|----------------|
| outer | `N − 2k + 2`   | `2k − 1`       |
| inner | `2k`           | `N − 2k + 1`   |

Print double-sided (flip on **short edge**), nest the sheets, fold along the
spine and staple — pages 1..N read in order. Pages beyond the original count
are left blank.

## Layout options

- **Sheet size** — `Auto` keeps the input page size and just doubles the
  width, so an A5 input becomes an A4-landscape sheet with no scaling. Or
  pick A4 / A3 / Letter / Tabloid landscape explicitly. The app picks a
  sensible default on load (A4-landscape for A4 input; otherwise auto).
- **Page placement** — `Fit` (letterbox), `Fill` (crop), or `Original`
  (1:1, centered).
