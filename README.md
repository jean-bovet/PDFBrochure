# PDFBrochure

A small SwiftUI macOS app that turns any PDF into a brochure ready for
saddle-stitch printing ‚Äî two input pages laid out per sheet, in the order
needed for double-sided printing and folding.

The flow is two screens:

1. **Drop** ‚Äî drag a PDF onto the window (or click to pick one).
2. **Layout + preview** ‚Äî pick a sheet size and page placement; the imposed
   PDF re-renders in a live preview. One click writes
   `<input>-brochure.pdf` next to the source (auto-suffixed `-2`, `-3`, ‚Ä¶ if
   the name is taken ‚Äî never overwrites an existing file).

A small headless `pdfbrochurecli` target is built from the same imposition
code for scripting and tests.

## Build & run

```sh
brew install xcodegen           # one-time
xcodegen generate               # creates PDFBrochure.xcodeproj from project.yml
open PDFBrochure.xcodeproj      # ‚Ä¶and hit Run, or:

xcodebuild -scheme PDFBrochure    -configuration Debug   build
xcodebuild -scheme pdfbrochurecli -configuration Release build
```

The built app lives under
`~/Library/Developer/Xcode/DerivedData/PDFBrochure-*/Build/Products/`.

> Don't pass `-derivedDataPath` to point at a folder under iCloud-managed
> directories like `~/Desktop` or `~/Documents`: the iCloud file-provider
> attaches xattrs that make codesign refuse to sign the resulting `.app`.

### Headless usage

```
pdfbrochurecli <input.pdf> <output.pdf> [auto|a4|a3|letter|tabloid] [fit|fill|original]
```

## How the imposition works

For an N-page document padded up to a multiple of 4, sheet *k* (1-indexed)
holds:

| side  | left half      | right half     |
|-------|----------------|----------------|
| outer | `N ‚àí 2k + 2`   | `2k ‚àí 1`       |
| inner | `2k`           | `N ‚àí 2k + 1`   |

Print double-sided (flip on **short edge**), nest the sheets, fold along the
spine and staple ‚Äî pages 1..N read in order. Pages beyond the original count
are left blank.

## Layout options

- **Sheet size** ‚Äî `Auto` keeps the input page size and just doubles the
  width, so an A5 input becomes an A4-landscape sheet with no scaling. Or
  pick A4 / A3 / Letter / Tabloid landscape explicitly. The app picks a
  sensible default on load (A4-landscape for A4 input; otherwise auto).
- **Page placement** ‚Äî `Fit` (letterbox), `Fill` (crop), or `Original`
  (1:1, centered).
